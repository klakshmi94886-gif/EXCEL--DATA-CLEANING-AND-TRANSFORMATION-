# 🧹 Excel Data Cleaning and Transformation Project

## 🔍 Project Overview
Data is rarely clean when first extracted from transactional systems. This project focuses on the end-to-end data preparation workflow—identifying anomalies, handling missing values, standardizing formats, and transforming raw records into a highly structured database ready for business intelligence tools. 

Using **Excel Assignment 2 - Data Cleaning and Transformation (2).xlsx**, this repository demonstrates the application of logical filtering, advanced sorting, duplicate elimination, and formatting techniques to establish a reliable "single source of truth."

---

## 🛠️ Technical Skills & Excel Functions Used

*   **Data Hygiene & Deduplication:** Identified, isolated, and permanently removed duplicate transactional entries while preserving historical integrity.
*   **Missing Value Imputation:** Applied conditional filters to address blank cells, ensuring data completeness without introducing analytical bias.
*   **Format Standardization:** Standardized date formats, numeric figures, and text casing to ensure uniformity across all attributes.
*   **Logical Auditing:** Leveraged advanced filtering, structural sorting, and data validation techniques to audit quality bounds across columns.

---

## 🎯 Steps Followed in Data Cleaning & Transformation

1.  **Structural Inspection:** Audited the initial dataset footprint to detect inconsistent columns, trailing whitespaces, and broken entries.
2.  **Duplicate Isolation:** Used Excel’s `Remove Duplicates` operational tools alongside conditional formatting rules to flags repeating records.
3.  **Handling Nulls & Blanks:** Isolated missing elements systematically, applying relevant business rules to either populate or remove incomplete entries.
4.  **Data Type Alignment:** Converted text-formatted numbers back to true numeric fields and applied uniform currency and short-date masking.
